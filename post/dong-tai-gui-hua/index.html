动态规划

2021-08-21

问题
我们正在玩一个猜数游戏，游戏规则如下：
我从 1 到 n 之间选择一个数字，你来猜我选了哪个数字。
每次你猜错了，我都会告诉你，我选的数字比你的大了或者小了。
然而，当你猜了数字 x 并且猜错了的时候，你需要支付金额为 x 的现金。直到你猜到我选的数字，你才算赢得了这个游戏。

思路
区间dp
极小化极大
类似矩阵乘法的最小次数，石子游戏，戳气球，最长回文子序列，扔鸡蛋
支付的总金额是猜对之前的总的尝试数字之和

子问题：在区间[i,j]，猜对数字，需要支付的总金额
状态转移方程：
我们可以枚举i到j之间每个数，作为猜测的数字
如果猜测的数字是k，分为三种情况
猜对，此时为0
猜小，k+dp[k+1][j]
猜大，k+dp[i][k-1]
三种情况取最大，所有数字取最小

初始化为最大值

遍历顺序
可以选择主对角线，次对角线顺序,枚举区间长度和区间左端点
可以选择从下往上，从左往右，也可以选择从左往右，从下往上

代码

class Solution {
public:
    int getMoneyAmount(int n) {

        // 区间dp
        // 极小化极大
        // 类似矩阵乘法的最小次数，石子游戏，戳气球，最长回文子序列，扔鸡蛋
        // 支付的总金额是猜对之前的总的尝试数字之和

        // 子问题：在区间[i,j]，猜对数字，需要支付的总金额
        // 状态转移方程：
        // 我们可以枚举i到j之间每个数，作为猜测的数字
        // 如果猜测的数字是k，分为三种情况
        // 猜对，此时为0
        // 猜小，k+dp[k+1][j]
        // 猜大，k+dp[i][k-1]
        // 三种情况取最大，所有数字取最小

        // 初始化为最大值

        // 遍历顺序
        // 可以选择主对角线，次对角线顺序,枚举区间长度和区间左端点
        // 可以选择从下往上，从左往右，也可以选择从左往右，从下往上

        vector<vector<int>> dp(n+1,vector<int>(n+1,INT_MAX));

        for(int i=0;i<=n;i++)    // 初始化，只有一个数不用尝试
        dp[i][i]=0;

        for(int i=0;i<n;i++)
        dp[i+1][i]=0;

        for(int len=2;len<=n;len++)
        for(int i=1;i+len-1<=n;i++){
            int j=i+len-1;

            for(int k=i;k<=j;k++)
            dp[i][j]=min(dp[i][j],max(dp[i][k-1]+k,k!=n?dp[k+1][j]+k:0));
        }

        return dp[1][n];
    }
};
